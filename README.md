
## Age Calculator Pro

Advanced web-based Age Calculator built using HTML, CSS, and JavaScript.

## 🌐 Live Demo
https://nadia-7-gm.github.io/CodeAlpha_AgeCalculator/

## Features
- Real-time age calculation
- Years, months, days
- Time lived (hours, minutes, seconds)
- Next birthday countdown
- Zodiac sign
- Compare two ages (optional)
- Responsive design
- Dark/Light mode
- Splash screen

## Tech Stack
- HTML
- CSS
- JavaScript (DOM + Date API)
